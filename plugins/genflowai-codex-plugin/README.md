# GenflowAI Plugin Package

This directory is the actual Codex plugin package referenced by `.agents/plugins/marketplace.json`.

It contains:

- `.codex-plugin/plugin.json` for Codex plugin metadata
- `.mcp.json` for the bundled GenflowAI MCP server
- `scripts/genflowai-mcp.mjs` for public marketplace tools
- `skills/genflowai-templates/SKILL.md` for Codex workflow guidance
